Instalando o docker

> ssh root@ip da máquina
> cat /etc/*release* (para ver a release do ubuntu)
- site oficial do Docker (procurar o script de download e instalação para ubuntu)
- executar o script
> docker version (ver a versão do docker)
> systemctl status docker

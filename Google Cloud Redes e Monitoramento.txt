GOOGLE CLOUD REDES E MONITORAMENTO

REDES VPC

Rede de Nuvem Privada Virtual

Similar a uma rede física. Implementada dentro do projeto.
É criada uma VPC default junto com o projeto.
Os produtos de infraestrutura se conectam a esta rede virtual.
É possível criar uma Cloud VPN da minha rede local para a VPC.
=================

INFRAESTRUTURA DE REDE GLOBAL

Quando criamos uma VPC ela é conectável à rede global da Google Cloud.
